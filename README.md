# <font color=#70C6B0>应用简介</font>
大家好，这是我开发的一个随机密码生成器，基于 ***.NET 6.0 控制台*** 框架，欢迎大家试用。

### <font color=#87CEEB>算法说明</font>
1. 获取字符池，包含所有ASCII表中可用的字符，如数字、字母、符号（不包含空格、DEL等）。
2. 字符池排列顺序：数字、字母、符号（默认字母为必须，因此放在中间）。
3. 字符类型采用了枚举（Flags）进行判定。
